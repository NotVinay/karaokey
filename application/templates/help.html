Frequently Asked Questions

How to use the application?

1. Upload music file.
2. Request Separation and wait for result.
3. Check and Save the separation.

Which files does karaokey support?

It supports almost all audio file types, here is the full list of files types supported.

  • .aac
  • .aif
  • .au
  • .flac
  • .gsm
  • .mp3
  • .ogg
  • .ra
  • .wav
  • .wma
  • .wav
  • .m4a

How long will the file i upload be saved on the system?

The file uploaded by the you will be kept for 1 to 2 days. User will be able to access the results of the separation for 1 day. After a day the results will no longer be available and you will have to perform the separation again.

How the system uses the file i upload?

The file you upload will not be used by the system for any personal use, it will only process the file through the model to generate the predicted results. The file will not be published anywhere on the site or be available to any other user.

Is the file shared with any other user or system?
