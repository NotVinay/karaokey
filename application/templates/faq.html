Frequently Asked Questions

How to use the application?

1. Upload music file.
2. Request Separation and wait for result.
3. Check and Save the separation.

How long the uploaded files are kept on the system?

The files uploaded by the you will be kept for 1 to 2 days. User will be able to access the results of the separation for 1 day. After a day the results will no longer be available and you will have to perform the separation again.

How the system uses the file i upload?

The file you upload will not be used by the system for any personal or commercial use, it will only process the file through the model to generate the predicted results. The file will not be published anywhere on the site or be available to any other user.

Is the file shared with any other user of system?
